Instructions:



This is a turn-based strategy game, which uses a mode-based click-and-click interface. Nearly everything in the interface except the map itself has tooltips that pop up if you hold your mouse over one of them. Click on a button at the top to set the mode, then click one or two tiles (depending on the mode) on the map to set where to apply the action. If the action is building, select what to build after you select the tile to build on.

The orange and black circles on the map represent units; use the Info mode to inspect them. Black units belong to player #1, while orange units belong to player #2.
